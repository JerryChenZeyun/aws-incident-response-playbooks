# AI-Powered Incident Response Automation
This README relates specifically to the steering documents that turn an incident-response process into an operator-driven automation investigation.

## Usage

These steering documents are written in markdown to facilitate editing and consumption into a variety of user systems.

Invoke using the following syntax in your IDE:

```Kiro
#steering-irp-<playbook0name>.md
```
